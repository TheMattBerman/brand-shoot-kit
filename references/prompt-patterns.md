# Prompt Patterns

Use direct operational structure:

1. shot purpose + channel + ratio
2. exact product description and preservation clauses
3. scene and composition controls
4. lighting and depth constraints
5. negative constraints
6. reroll instruction tied to known failure

## Template

```markdown
Use case: <PDP/Lifestyle/...>
Ratio: <1:1/4:5/16:9/9:16>
Prompt: Create a ...
Preserve: exact shape, label placement, logo, count
Avoid: no extra logos, no fake text, no malformed hands
Reroll: if <failure>, adjust <specific correction>
```
