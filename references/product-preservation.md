# Product Preservation Guide

Product fidelity is the main quality gate for ecommerce imagery.

## Never Change

- brand name spelling
- core package geometry
- key label claims and warnings
- cap/nozzle/pump type when visible
- product count for single-item shots

## Allowed Variation

- lighting direction and intensity
- camera distance and angle
- background/set context
- supporting props (non-competing)
- hand/model framing

## High-Risk Categories

- transparent/glass packaging
- technical industrial labels
- legal/compliance-heavy products
- jewelry and tiny-detail luxury items

For high-risk categories, prefer planning output + conservative shot prompts.
