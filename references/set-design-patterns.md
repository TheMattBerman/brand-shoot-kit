# Set Design Patterns

Choose set language that supports product function and price tier.

## Set Controls

- surface: stone/wood/tile/fabric/metal
- depth: flat sweep vs layered depth
- clutter: low for premium, moderate for lived-in
- light: clinical bright vs warm editorial
- prop role: support product story, never steal focus

## Good Default Sets

- bathroom vanity
- kitchen counter
- desk/work bag
- bedside table
- gym locker/bag
- clean studio sweep

## Avoid

- random prop piles
- impossible scale relationships
- stock-photo cliches detached from product use
