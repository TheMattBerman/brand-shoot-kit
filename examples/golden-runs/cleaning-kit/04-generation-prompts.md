# Generation Prompts

## Shot 01 - Clean front kit hero

**Use case:** PDP
**Aspect ratio:** 1:1

**Prompt:**
Create a 1:1 ecommerce image for Clean Home Essentials Kit (BrightNest) optimized for PDP. Shot intent: Clean front kit hero. Scale guidance: clean hero framing; product fills 62-75% with immediate label readability. Human guidance: no human subject required; keep focus on pack fidelity and commerce clarity. Context guidance: controlled ecommerce set with minimal props and clean background separation. Brand tone: brand-consistent ecommerce. Scene direction: Use sink, countertop, and cloth/brush props with bright realistic daylight. Keep kit components and packaging copy clearly readable. Evidence anchors from source: label. Must preserve exactly: bottle/refill geometry, front label hierarchy, kit component count. Never change: active ingredient statements, size/volume callouts, brand name. Watch distortion risks: trigger geometry deformation, label text drift, missing kit components. Keep props supportive, never dominant.

**Negative constraints:**
- no altered logo or label text
- no fake ingredient/claim badges
- no malformed hands or impossible anatomy
- no product count mismatch for intended SKU
- no prop occlusion hiding the primary label

**Reroll if failed:**
If label legibility, pack geometry, or product count fails, reroll with flatter camera angle, cleaner background, and tighter product framing.
